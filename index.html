Overview

Raw Dog is a standalone desktop GUI for parsing, processing, and visualizing time-series data. Built for hobbyists, engineers and researchers, it enables a raw-to-plot workflow with no coding required. It follows a glass-box philosophy, exposing the underlying Python code for inspection and customization.

Key Features

  • Instant workflow: Load → Parse → Process → Visualize
  • Signal operations: filter, normalize, resample, and more
  • Comparison tools: Bland-Altman, correlation, and error histograms
  • Customizable plots: Multi-subplot layouts with stylization options
  • Transparent backend: Inspect, edit, and run custom Python scripts

Install Instructions

Follow these steps to get started — no coding required.

  1. Download the correct .zip file for your system (see buttons above).
  2. Unzip the file:
    • macOS: Double-click the downloaded .zip file to extract it.
    • Windows: Right-click the downloaded .zip file and select "Extract All..." to extract it.
    This will create a folder containing the application.
  3. Open the app: Inside the folder, double-click the file named RawDog.app or GUI8.app (on macOS), or RawDog.exe (on Windows) to launch the tool.
  4. The app may take 1–2 minutes to launch the first time. This is expected — please be patient.
  5. Start with the included test data. Large datasets may take longer to load or may appear unresponsive at first — give it a moment.

Compatibility Notes:

  • macOS: Tested on macOS 12 and above. Compatible with both Intel and Apple Silicon (M1/M2) machines.
  • Windows: Tested on Windows 10 (64-bit). Compatibility with other versions (e.g. Windows 11 or older) has not been verified.
